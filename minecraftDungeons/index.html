Minecraft Dungeons Modding Tutorial

Warning

This Tutorial is for Windows 10/11 Only, I cannot garenty that it will or won't work on other operating systems.

Things We will be using.

  1. MCD Save Edit
  2. Minecraft Dungeons
  3. UWP Dumper (Only if you have microsoft store version of game.)

What to do if you bought the game on the microsoft store.

If you bought the game on the microsoft store, there is a few more things you will have to do. First download UWP Dumper. Make sure you dowload the right version for your Windows OS, (32x/64x).
In my case I have the 64x version so I download that. Next extract the zip folder that downloads before you run the exe file. Now open minecraft dungeons. Once you have ran the UWPDumper file you should see a big list with all of the UWP programms running on your computer. Look for one called Dungeons.exe then put the number to the left of it into the text nox at the bottom, if you cant find one, thats okay. Open task manager then find minecraft dungeons, click goto details then find the process ID, put that into UWPDumper. Once you complete those steps it should download a folder called DUMP. To veiw it goto C:\Users\Username\AppData\Local\Packages\Microsoft.Lovika\TempState. When you open MCDsave Edit and it asks you where the .pak files are select the DUMP folder.
